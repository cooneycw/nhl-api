# NHL API

A consolidated NHL data API project, combining the best features from multiple existing implementations.

## Project Status

Initial setup - project scaffolding in place.

## Goals

- Consolidate NHL data pulling from existing cooneycw repos
- Provide a clean, unified API for NHL statistics
- Future integration with oneNinety dashboard

## Setup

```bash
# Clone the repository
git clone git@github.com:cooneycw/nhl-api.git
cd nhl-api

# Create conda environment (when environment.yml exists)
conda env create -f environment.yml
conda activate nhl-api
```
